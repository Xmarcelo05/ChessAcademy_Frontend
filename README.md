# 🎯 ChessAcademy - Landing Page

Una landing page moderna, elegante y altamente interactiva para cursos de ajedrez.

## 📋 Características

✅ **Diseño Elegante**: Paleta de colores de ajedrez (blanco, negro, dorado)
✅ **Animaciones Fluidas**: Transiciones suaves y efectos visuales cautivadores
✅ **Totalmente Responsive**: Funciona perfectamente en mobile, tablet y desktop
✅ **Secciones Completas**:
   - Hero con tablero de ajedrez interactivo
   - Introducción al ajedrez
   - Técnicas fundamentales
   - Años de experiencia
   - Progreso y estadísticas
   - Módulos de aprendizaje
   - Footer con llamada a acción

✅ **Archivo único**: Todo en un HTML con CSS y JavaScript integrados
✅ **Sin dependencias externas**: No requiere librerías o frameworks adicionales
✅ **Amigable para principiantes**: Interfaz intuitiva y acogedora

## 🚀 Cómo Usar

### Opción 1: Abrir directamente
1. Descarga el archivo `index.html`
2. Haz doble clic para abrirlo en tu navegador
3. ¡Listo! La página está lista para usar

### Opción 2: Servir localmente (recomendado)
```bash
# Con Python 3
python -m http.server 8000

# Con Node.js
npx http-server

# Con Live Server (VSCode extension)
# Click derecho en index.html > Open with Live Server
```

Luego accede a `http://localhost:8000`

## 📱 Navegación

## 📝 Personalización

Para personalizar la página, abre `index.html` con un editor de texto y modifica:

### Cambiar colores:
Busca la sección `:root` y modifica los valores CSS:
```css
--color-primary: #1a1a1a;
--color-secondary: #f5f1e8;
--color-accent: #d4a574;
```

### Cambiar textos:
Busca el HTML y reemplaza los textos en las secciones (`<h1>`, `<p>`, etc.)

### Cambiar iconos:
Busca los símbolos como `♛`, `🎯`, `♟️`, etc. y reemplázalos con otros
